# Custom Fighting Game Gamepad
ストリートファイター６用の自作ゲームパッドです。  
スティックの上入力を無効化できるものが欲しくて作りました。  
Raspberry Pi PicoにGP2040-CEをインストールして使います。  
素人設計なので拙い部分が多々あると思います。データを利用する場合は自己責任でお願いします。
